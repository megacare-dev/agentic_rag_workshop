  • 🖼️ Gemini Multimodal — แปลง PDF ด้วย AI
  • 🇹🇭 Thai Tokenization — ตัดคำภาษาไทย
  • 📐 Embedding — แปลงข้อความเป็น Vector
  • 🔎 Hybrid Search — Dense + Sparse
  • 💾 Qdrant — Vector Database + Indexing

Building Agents

  • 🔧 Tools — Function Tools + Gemini Params
  • 📚 RAG Agent — Agent ค้นหาจาก VectorDB
  • 👥 Multi-Agent — Sequential / Parallel / Loop
  • 🧠 Session Memory — จำบทสนทนา

Evaluation

  • ⚡ A/B Experiment — เปรียบเทียบ configs
